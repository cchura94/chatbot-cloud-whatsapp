## Levantar el proyecto

### Instalar las dependencias del proyecto
```
npm install
```
- Este comando descarga e instala todos los paquetes que necesita mi proyecto de node

### Actualizar variables de entorno 
- Se necesita asignar parametros en el archivo (.env) 
```
WHATSAPP_ACCESS_TOKEN=
WHASTAPP_PHONE_NUMBER_ID=
```
- Los datos poder obtener desde https://developers.facebook.com

### Levantar el proyecto de node
```
node src/index.js
```
